# Stretch Break Browser Extension

Chromium MV3 extension scaffold using WXT + React + TypeScript.

## Install

```bash
pnpm install
```

## Run development mode

```bash
pnpm dev
```

Then load the generated unpacked extension from `.output/chrome-mv3-dev` in Chromium.

## Run tests

```bash
pnpm test
```

## Build production package

```bash
pnpm build
```
